# NutriSoft

O NutriSoft é uma aplicação desktop desenvolvida em Java (Swing) com foco em cálculos e orientações nutricionais básicas. O sistema permite ao usuário calcular indicadores de saúde e receber recomendações de forma simples e didática.

O projeto foi estruturado utilizando o NetBeans e o sistema de build Ant, sendo indicado para fins educacionais e acadêmicos.


## ⚙️ Funcionalidades

* Cálculo de IMC (Índice de Massa Corporal)

* Cálculo de Gasto Calórico

* Recomendações nutricionais básicas

* Interface gráfica amigável (Java Swing)
